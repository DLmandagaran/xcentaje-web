# Xcentaje Web

🌐 **Web en Vivo:** [https://dlmandagaran.github.io/xcentaje-web/](https://dlmandagaran.github.io/xcentaje-web/)

## 🎯 Objetivo del Proyecto

Este repositorio aloja la versión estática (HTML, CSS y JavaScript) de **Xcentaje**, diseñada con un enfoque moderno, rápido y estético.

El objetivo principal de esta página es servir como un portfolio profesional y una "landing page" impactante para presentar los principales servicios de la empresa:
- **Data Science / Analytics:** Visualización avanzada y tableros interactivos (como la integración con Power BI).
- **AI & ML:** Modelado predictivo y automatización de procesos.
- **Public Data:** Extracción, procesamiento y análisis de datos públicos disponibles en el mercado.

## 🚀 Tecnología

La página está construida con tecnologías frontend nativas (HTML5, Vanilla CSS avanzado y JS) sin requerir bases de datos ni procesamiento backend (PHP), lo que permite que sea alojada gratuitamente y a máxima velocidad utilizando **GitHub Pages**.
